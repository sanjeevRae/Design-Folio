# Design Folio

## Booking Form Email Setup

The booking form sends emails through EmailJS using environment variables from `.env`:

- `VITE_EMAILJS_SERVICE_ID`
- `VITE_EMAILJS_TEMPLATE_ID`
- `VITE_EMAILJS_PUBLIC_KEY`

If EmailJS returns `Gmail_API: Invalid grant`, reconnect the Gmail account in the EmailJS dashboard. That error is provider-side, not a code issue in this app.
